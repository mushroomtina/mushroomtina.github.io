常用hexo写博客步骤

参考：https://github.com/peiying/hexo/blob/master/source/_posts/%E6%9C%80%E9%9D%A0%E8%B0%B1%E7%9A%84windows%E4%B8%8B%E6%90%AD%E5%BB%BA%E7%8B%AC%E7%AB%8B%E5%8D%9A%E5%AE%A2GitHub%20Pages%E4%B8%8EHexo%E7%9A%84%E6%95%99%E7%A8%8B.md

  1. 进入文件夹D:\自学\Blog，右键打开git bash。在里面输入命令。
  2. hexo new “blog post name”: 创建新博客
  3. hexo g
  4. hexo s: 预览网页。通过localhost:4000。ctrl+c关闭。
  5. hexo d：部署并上传到github。
  6. 打开github个人网站https://mushroomtina.github.io/。如已经打开过需要通过shift+F5刷新，才能加载成功。
